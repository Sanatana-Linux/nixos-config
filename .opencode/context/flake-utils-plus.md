# Flake-Utils-Plus (FUP)

> Source: https://github.com/gytis-ivaskevicius/flake-utils-plus | License: MIT | v1.5.1

Library abstraction to painlessly generate NixOS flake configurations. Extends [flake-utils](https://github.com/numtide/flake-utils). Design goal: keep it simple, no fluff.

## Key Features

- `lib.mkFlake { ... }` — clean flake generation
  - `nix.generateRegistryFromInputs` — auto-generate `nix.registry`
  - `nix.generateNixPathFromInputs` — auto-generate `nix.nixPath`
  - `nix.linkInputs` — symlink inputs to `/etc/nix/inputs`
  - Multiple `nixpkgs` references (channels)
  - `hostDefaults` — shared config between hosts
  - `outputsBuilder` — clean package/app export
  - `sharedOverlays` — overlays applied across all channels
- `lib.exportModules [ ./a.nix ./b.nix ]` — namespace modules
- `lib.exportOverlays channels` — namespace overlays
- `lib.exportPackages self.overlays channels` — cacheable cross-flake packages
- `pkgs.fup-repl` — kick-ass REPL (`repl` / `repl /path/to/flake.nix`)

## Full Configuration

```nix
{
  let
    inherit (builtins) removeAttrs;
    mkApp = utils.lib.mkApp;
    pkgs = self.pkgs.x86_64-linux.nixpkgs;
  in flake-utils-plus.lib.mkFlake {

    inherit self inputs;                # REQUIRED

    supportedSystems = [ "x86_64-linux" ];

    # --- channels ---
    channelsConfig = { allowBroken = true; };
    sharedOverlays = [ nur.overlay ];
    channels.<name>.input = nixpkgs;    # autogenerated from inputs by default
    channels.<name>.config = { allowUnfree = true; };
    channels.<name>.patches = [ ./someAwesomePatch.patch ];
    channels.<name>.overlaysBuilder = channels: [
      (final: prev: { inherit (channels.unstable) neovim; })
    ];

    # --- hostDefaults ---
    hostDefaults.system = "x86_64-linux";
    hostDefaults.modules = [ ./module.nix ./module2 ];
    hostDefaults.channelName = "unstable";
    hostDefaults.extraArgs = { inherit utils inputs; foo = "foo"; };

    # --- hosts ---
    hosts.<hostname>.system = "aarch64-linux";
    hosts.<hostname>.channelName = "unstable";
    hosts.<hostname>.extraArgs = { abc = 123; };
    hosts.<hostname>.specialArgs = { thing = "abc"; };
    hosts.<hostname>.modules = [ ./configuration.nix ];
    hosts.<hostname>.output = "darwinConfigurations";   # also supports darwin
    hosts.<hostname>.builder = nix-darwin.lib.darwinSystem;

    # --- outputsBuilder ---
    outputsBuilder = channels: {
      apps = {
        custom-neovim = mkApp { drv = fancy-neovim; exePath = "/bin/nvim"; };
      };
      packages = { inherit (channels.unstable) package-from-overlays; };
      defaultApp = mkApp { drv = channels.nixpkgs.firefox; };
      defaultPackage = channels.nixpkgs.neovim;
      devShell = channels.nixpkgs.mkShell { name = "devShell"; };
    };

    # --- passthrough ---
    checks.x86_64-linux.someCheck = pkgs.hello;
    packages.x86_64-linux.somePackage = pkgs.hello;
    overlay = import ./overlays;
    abc = 132;
  };
}
```
