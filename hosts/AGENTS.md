# hosts/

## Purpose

## For AI Agents

### Working In This Directory
- Each host **must** import only the user module(s) it needs (e.g., `../../modules/nixos/users/tlh.nix`) — never import all user modules
- `hardware-configuration.nix` is auto-generated by `nixos-generate-config` — do not edit manually
- Host `default.nix` sets `modules.<category>.<name>.enable = true` for modules it uses
- Add `pkgs` to the function arguments if referencing custom packages from `pkgs/`

### Common Patterns
- Host config receives `{inputs, lib, outputs, pkgs, ...}` from flake.nix
- Module enables go in a `modules` attribute set at top level
- `system.stateVersion` must be set per host
