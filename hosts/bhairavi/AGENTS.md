# bhairavi

## Purpose
Virtual machine host — QEMU/KVM guest running AwesomeWM on modesetting driver, user tlh. Lightweight VM config with Docker (rootless, no NVIDIA), no hardware-specific modules (no bluetooth, TPM, Logitech, Lenovo, Android), no power management, comprehensive development packages, and full security stack (firewall, SSH, fail2ban, doas).
